GITHUB ACTIONS İLE APK (ÖNERİLEN - EN GARANTİLİ)
=================================================
Colab zaten 2 kere hata verdi. GitHub Actions çok daha kararlı.

1. https://github.com/ adresinde bir repo oluştur (ör: tupgaz-mobile)
2. Aşağıdaki kodları sırayla çalıştır:

   cd C:\Users\212\Desktop\tüp
   git init
   git add .
   git commit -m "ilk"
   git remote add origin https://github.com/KULLANICI_ADIN/tupgaz-mobile.git
   git push -u origin main

3. GitHub'da repo sayfasına git → Actions sekmesi
4. Soldan "Build APK" workflow'una tıkla → "Run workflow" → yeşil buton
5. ~30 dk bekle, bitince workflow sayfasında "Artifacts" bölümü
6. tupgaz-mobile-apk.zip indir → içinden .apk'yı çıkar → WhatsApp'tan gönder
